# Admin Control Portal: User Guide

## Introduction

The Admin Control Portal is your central command center for managing the authentication system. From here, you can:

- View and approve new user registrations
- Monitor active members
- Track login history
- Manage system settings
- View analytics and reports

---

## Getting Started

### Accessing the Portal

1. Open your web browser
2. Navigate to the Admin Portal URL (e.g., `https://admin.yourcompany.com`)
3. You'll see the login page

### Logging In

1. Enter your **username** (not email)
2. Enter your **password**
3. Click **Sign In**

If you've forgotten your password, click "Forgot Password" and follow the instructions.

---

## Dashboard Overview

After logging in, you'll see the main dashboard with:

### Statistics Cards

- **Total Members** - Number of approved, active users
- **Active Invitations** - Pending registration requests
- **Media Files** - Uploaded documents and images
- **Growth Rate** - Membership growth percentage

### Recent Activity

A timeline showing the latest events:

- New member joins
- Invitations sent
- Documents uploaded
- Login activity

---

## Managing Users

### Viewing Pending Registrations

1. Click **Invitations** in the sidebar
2. You'll see a list of users waiting for approval
3. Each entry shows:
   - Name
   - Email
   - Username
   - Registration date

### Approving a User

1. Go to the Invitations page
2. Find the user you want to approve
3. Click **Approve**
4. (Optional) Add approval notes
5. Click **Confirm Approval**

The user will be:

- Moved to active members
- Assigned a unique auth key
- Sent an approval email notification

### Rejecting a User

1. Go to the Invitations page
2. Find the user you want to reject
3. Click **Reject**
4. Enter a rejection reason (required)
5. Click **Confirm Rejection**

The user's application will be archived with the rejection reason.

## Viewing Members

### Active Members List

1. Click **Members** in the sidebar
2. View all approved, active users
3. Information shown:
   - Username
   - Email
   - Full Name
   - Status (Active/Inactive)
   - Last Login Date

### Member Details

Click on any member row to see:

- Complete profile information
- Login history
- Services they've accessed
- Account creation date
- Total login count

## Login History

### Viewing All Logins

1. Click **Analytics** or navigate to Admin section
2. Select **Login History**
3. View chronological list of all login events

### Information Tracked

For each login:

- User name
- Service accessed
- Login time
- Session duration
- IP address (if available)

### Filtering History

Use the filters to narrow down:

- By specific user
- By specific service
- By date range

## System Status

### Checking Operating Hours

The system enforces operating hours. To check:

1. Look at the system status indicator (usually in the header)
2. Green = System Open
3. Yellow = Warning (closing soon)
4. Red = System Closed

### Understanding Status Messages

| Status | Meaning |
|--------|---------|
| System Open | Normal operations |
| Closing in X minutes | Save your work |
| System Closed | No logins allowed |

## Troubleshooting

### "Session Expired" Message

Your login session has timed out. This happens after 30 minutes of inactivity.
**Solution:** Log in again.

### Can't See Pending Users

You may not have admin privileges.
**Solution:** Contact your super admin to upgrade your account.

### Changes Not Saving

Check your internet connection. If persistent, try:

1. Refresh the page
2. Log out and log back in
3. Clear browser cache

### Dark Mode Not Working

Clear your browser's local storage:

1. Open Developer Tools (F12)
2. Go to Application tab
3. Clear Local Storage
4. Refresh the page

## Security Best Practices

1. **Never share your password**
2. **Log out when finished** (especially on shared computers)
3. **Use a strong password** (mix of letters, numbers, symbols)
4. **Report suspicious activity** immediately
5. **Don't bookmark the logged-in session** - always log in fresh
